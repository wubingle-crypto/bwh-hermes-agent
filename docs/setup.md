# Self-Hosting Guide | 自行部署

## Prerequisites
- Python 3.11+
- Node.js 18+
- Telegram Bot Token (from @BotFather)
- DeepSeek API Key
- BSC Wallet with BEP-20 USDT support

## Quick Start
```bash
# Clone
git clone https://github.com/YOUR_USER/bwh-hermes-agent
cd bwh-hermes-agent

# Install dependencies
pip install -r requirements.txt

# Configure
cp .env.example .env
# Edit .env with your tokens

# Run
python3 bot/bot.py
```
