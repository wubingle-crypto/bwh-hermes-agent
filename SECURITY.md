# Security Policy | 安全策略

## Payment Security | 支付安全
- All payments are verified on-chain via BSC
- Never share your private keys
- Only send BEP-20 USDT to the provided address
- Verify the wallet address on BscScan before sending

## Proxy Security | 节点安全
- All proxy traffic is TLS 1.3 encrypted
- No logs are kept of user browsing activity
- Nodes use randomized Cloudflare IPs for censorship resistance
